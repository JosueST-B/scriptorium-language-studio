# Scriptorium Website

Professional static website for Scriptorium, a translation, localization and documentation studio.

## Open locally

Open `index.html` in a browser, or use `abrir-scriptorium.bat`.

## Main files

- `index.html`: site structure and content
- `styles.css`: responsive institutional design
- `translations.js`: multilingual content dictionary for 12 languages
- `script.js`: mobile menu, language switcher, portfolio filters and quote message helper
- `assets/`: founder portrait and portfolio visuals
- `portfolio/`: downloadable sample files

## Notes

The portfolio items are presented as fictional samples. The site does not claim certified, sworn, official, medical, legal or academic advice.

## Language coverage

The interface includes EN, ES, DE, FR, PT, IT, RU, CS, ZH, JA, HE and AR, with right-to-left layout support for Hebrew and Arabic.
